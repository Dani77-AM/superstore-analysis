# Análisis Exploratorio y Visualización de Datos Comerciales

## Descripción
Análisis exploratorio del dataset Superstore Sales con más de 9,994 
transacciones comerciales. Se identifican patrones de rentabilidad 
por categoría, región y sub-categoría.

## Herramientas
- Python (pandas, matplotlib, seaborn)
- Power BI Desktop
- GitHub

## Insights Clave
- Technology tiene el margen más alto con 17.4%
- Furniture vende $741K pero solo genera 2.49% de margen
- Los descuentos mayores al 40% generan pérdidas en todas las categorías
- La región Central tiene el margen más bajo con 7.92%
- Tables genera pérdidas de $17,000 — el peor producto del catálogo
